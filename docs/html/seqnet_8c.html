◆ SeqNet_loop()

SeqNet_Out SeqNet_loop ( const bool condition_active)

Executes the current instruction and advances or jumps the PC.

This function fetches the current instruction from memory (at PC), interprets it field-by-field, returns the requested output structure, and updates PC based on the condition_active flag.

Parameters
